# 演员评论家
演员评论家算法是强化学习中一类结合了基于策略（Policy-Based）和基于价值（Value-Based）方法的算法。
它由两个核心组件构成：演员（Actor） 负责学习策略，评论家（Critic） 负责评估当前策略的好坏。通过两者的协同工作，Actor-Critic 能够在连续动作空间和高维状态空间中高效学习，同时降低策略梯度的方差

## 为什么需要 Actor-Critic？
- 基于策略的方法（如 REINFORCE） 直接对策略进行梯度上升，但其梯度估计通常具有高方差，因为需要利用完整回报（Monte Carlo return）来计算，且没有利用价值函数作为基线来减少方差。

- 基于价值的方法（如 Q-learning） 通过学习价值函数间接得到策略，但在连续动作空间中难以处理，且容易陷入过估计。

Actor-Critic 结合了两者：Critic 通过学习价值函数为 Actor 提供低方差的梯度估计，同时 Actor 能够直接优化策略，适用于连续或高维动作空间。

## 核心思想
Actor（策略网络 $\pi_\theta(a|s)$）根据当前状态选择动作，并接收 Critic 的反馈来调整策略参数 $\theta$，目标是最大化累积奖励。

Critic（价值网络 $V_\phi(s)$ 或 $Q_\phi(s,a)$）估计状态值或动作值，为 Actor 提供优势函数（Advantage Function），即当前动作相对于平均水平的优势。  

优势函数定义为：
$ A(s,a) = Q(s,a) - V(s) $  
它衡量在状态 $s$ 下采取动作 $a$ 相对于平均表现的好坏。使用优势函数可以显著降低策略梯度的方差。

实际中，常使用时序差分误差（TD-error）作为优势函数的无偏估计：  
$ \delta_t = R_t + \gamma V(S_{t+1}) - V(S_t) $  
其中 $R_t$ 是即时奖励，$\gamma$ 是折扣因子。

## 算法流程
以最简单的单步 TD Actor-Critic 为例，每步更新流程如下：
1. 采样：在当前状态 $s$，Actor 根据策略 $\pi_\theta(\cdot|s)$ 采样动作 $a$，执行后得到奖励 $r$ 和下一状态 $s'$（以及回合结束标志 done）。
2. 计算 Critic 的 TD 目标：
如果 $s'$ 是终止状态，则 $V(s') = 0$。
否则，用当前 Critic 网络计算 $V(s')$（或使用目标网络）。
TD 目标：$y = r + \gamma V(s')$。
3. 计算 TD 误差（即优势估计）:$ \delta = y - V(s)$。
4. 更新 Critic：最小化 TD 误差的平方损失 $L_{critic} = \delta^2$，更新 Critic 参数 $\phi$。
5. 更新 Actor：使用优势函数 $\delta$ 作为权重，最大化策略的对数概率：$L_{actor} = -\log \pi_\theta(a|s) \delta$，更新 Actor 参数 $\theta$。

## 策略梯度的统一形式与不同权重方案

策略梯度的统一期望形式为：
$$
\nabla_\theta J(\theta) = \mathbb{E}_{\tau \sim \pi_\theta} \left[ \sum_{t=0}^T \Phi_t \nabla_\theta \log \pi_\theta(A_t|S_t) \right]
$$

其中权重 $\Phi_t$ 有四种典型选择，对应不同的策略梯度算法：

1.  $\Phi_t = G(\tau)$  
    对应**最简单的策略梯度法**，使用整条轨迹的总回报作为权重。

2.  $\Phi_t = G_t$  
    对应 **REINFORCE** 算法，使用从时刻 $t$ 开始的折扣未来回报 $G_t$ 作为权重。

3.  $\Phi_t = G_t - b(S_t)$  
    对应**带基线的REINFORCE**算法，引入基线 $b(S_t)$ 来降低梯度估计的方差，同时保持无偏性。

4.  $\Phi_t = R_t + \gamma V(S_{t+1}) - V(S_t)$  
    对应 **Actor-Critic** 算法，用优势函数（即时奖励+折扣下一个状态价值-当前状态价值）作为权重，是更高效的梯度估计方式。
